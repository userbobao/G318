# G318 摩旅路书地图

这是一个用于 GitHub Pages 发布的 G318 武汉到拉萨摩旅路书地图。

发布入口在：

```text
docs/index.html
```

地图页面在：

```text
docs/amap-jsapi/roadbook-map.html
```

发布前需要编辑：

```text
docs/amap-jsapi/config.local.js
```

填写高德 Web 端 JSAPI Key 和 JSAPI 安全密钥，并在高德控制台把 GitHub Pages 域名加入白名单。

详细步骤见：

```text
docs/DEPLOY_GITHUB_PAGES.md
```
